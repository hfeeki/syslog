Description

The event handler to be attached to the syslog_logger event manager. This handler will convert the incoming messages to syslog_reports which will then be converted to Syslog packets using the configured protocol backend and finally send the binaries over the wire.

Protocol backends must implement the syslog_error_h behaviour.
