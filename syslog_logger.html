Description

The main gen_event manager of the syslog application. All logged events will be directed through this process. This module does also define an internal event handler used for accelerate/decelerate message submission by toggling between synchronous and asynchronous gen_event notification.

Since the above idea was taken from basho's lager project this file was put under lager's license (Apache 2.0) and it's copyright was inherited. lagers implementation of the same feature can be found in the module lager_backend_throttle.erl.
